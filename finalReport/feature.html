Missing data

①　確認有無Missing data

sapply(Titanic, function(x) sum(is.na(x)))

PassengerId Survived Pclass Name Sex Age SibSp Parch Ticket Fare Cabin Embarked
0 0 0 0 0 177 0 0 0 0 687 2
發現Age、Cabin與Embarked三個欄位都有missing data，其中Cabin有687筆missing data。

②　Drop不必要欄位

●　PassengerId、Name、Ticket等欄位對於是否存活沒有影響， 且Cabin的missing data過多，因此選擇將這些columns刪除。
●　再來，將Embarked和Age的資料中有missing data的rows刪除
Titanic_train <- subset(Titanic, select = -c(PassengerId, Name, Ticket, Cabin))
Titanic_train <- Titanic_train[complete.cases(Titanic_train[,c("Embarked","Age")]),]

查看刪除missing data後，剩下多少資料
glimpse(Titanic_train)



刪除後的資料剩下712筆資料和8欄feature。
